# 🚀 Portfolio — Lucas Gomes do Amaral

Portfólio pessoal profissional construído com React + Vite, Tailwind CSS e Framer Motion.

## ✨ Stack

- **React 18** + **Vite 5** — estrutura moderna e build rápido
- **Tailwind CSS 3** — utilitários de estilo
- **Framer Motion** — animações avançadas e microinterações
- **Lenis** — scroll suave premium
- **Lucide React** — ícones limpos e modernos

## 🎨 Design

- Dark mode premium com paleta profunda (preto, azul prussiano, roxo)
- Tipografia: **Syne** (display) + **Outfit** (body) + **JetBrains Mono** (código)
- Glassmorphism sutil, partículas animadas no hero, grids de pontos
- Animações de entrada ao scroll, hover states com glow suave

## 🔧 Como rodar localmente

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## 🚀 Deploy

Pronto para deploy instantâneo em:
- **Vercel** — conecte o repositório e faça deploy em 1 clique
- **Netlify** — arraste a pasta `dist/` após `npm run build`
